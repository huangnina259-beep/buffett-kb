# Warren Buffett 年会文字记录提取工具

## 🎯 功能
- 自动提取 Berkshire Hathaway 年度股东大会的完整文字记录
- 清洗和格式化文本数据
- 保存为PDF格式，方便上传到NotebookLM
- 支持批量提取多年数据

## 📦 安装步骤

### 1. 安装依赖
打开命令行（CMD或PowerShell），运行：

```bash
pip install -r requirements.txt
```

### 2. 运行脚本

```bash
python buffett_meeting_scraper.py
```

## 🚀 使用方法

运行脚本后，会看到三个选项：

### 选项1: 仅提取2024年
- 最快速，只提取2024年的内容
- 适合快速测试

### 选项2: 提取所有年份
- 自动提取所有可用年份
- 适合建立完整档案

### 选项3: 提取指定年份范围
- 例如：2020-2024
- 适合按需提取

| 方法 | Token消耗 | 优缺点 |

## 🎯 上传到NotebookLM

1. 打开 [NotebookLM](https://notebooklm.google.com/)
2. 创建新笔记本
3. 上传生成的PDF文件
4. 开始分析和提问

## ⚠️ 注意事项

1. **网络连接**：确保能访问 buffett.cnbc.com
2. **请求频率**：脚本已设置2秒延迟，避免请求过快
3. **内容验证**：建议先提取一年测试效果
4. **备份机制**：如果PDF生成失败，会自动保存为TXT

## 🔧 可能的问题

### 问题1: 网站结构变化
如果提取失败，可能是网站更新了结构。可以：
- 手动检查网页源代码
- 修改脚本中的CSS选择器

### 问题2: 安装依赖失败
使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题3: 编码错误
脚本使用UTF-8编码，Windows用户应该没问题。
